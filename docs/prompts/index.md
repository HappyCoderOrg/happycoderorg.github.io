# 熱門 AI Prompt 提示詞

在 AI 時代，善用 Prompt 能大幅提升工作效率！本文收錄各職業領域最實用的 AI 指令模板，讓您直接複製使用，立即提升工作產出品質。

## 🎯 使用說明

### 📝 **如何使用這些 Prompt**
1. **複製模板**：直接複製需要的 Prompt 文字
2. **替換變數**：將 `[方括號]` 內容替換為您的具體資訊
3. **貼上使用**：在 ChatGPT、Claude、Gemini 等 AI 工具中使用
4. **調整優化**：根據結果調整 Prompt 內容

### ⚡ **提升效果的技巧**
- **明確目標**：清楚說明要達成的結果
- **提供背景**：給予充足的上下文資訊
- **指定格式**：明確要求輸出的格式和結構
- **設定角色**：讓 AI 扮演專業角色回答

## 📋 行政文書 Prompt

### 📧 **會議記錄整理**
```
請幫我整理這次會議的記錄，包含以下結構：

會議資訊：
- 時間：[會議時間]
- 參與者：[與會人員]
- 主題：[會議主題]

討論內容：
[貼上原始會議記錄或錄音逐字稿]

請整理成：
1. 重要決議事項
2. 待辦事項（負責人+期限）
3. 後續追蹤要點
4. 下次會議時間

請用清楚的條列式格式呈現。
```

### 📝 **公文撰寫**
```
請幫我撰寫一份正式的公文，內容如下：

公文類型：[函/簽/便箋等]
收件對象：[對象單位/人員]
主旨：[簡述目的]
具體內容：[詳細說明事項]
期望結果：[希望對方採取的行動]

請使用正式的公文格式，語氣要：
- 正式但不失禮貌
- 條理清晰
- 重點突出
```

### 📊 **報告摘要製作**
```
請幫我將以下長篇報告製作成摘要：

原始報告內容：
[貼上報告內容]

請整理成：
1. 執行摘要（3-5 句話）
2. 重要發現（3-5 點）
3. 關鍵數據
4. 建議行動方案
5. 風險提醒

摘要長度控制在 300 字以內，適合主管快速閱讀。
```

## 👥 人力資源 Prompt

### 📋 **職位描述撰寫**
```
請幫我撰寫一份吸引人的職位描述：

職位名稱：[職位名稱]
部門：[部門名稱]
工作地點：[地點]
薪資範圍：[薪資區間]

職位要求：
[列出技能、經驗、學歷要求]

公司特色：
[公司文化、福利、發展機會]

請撰寫包含：
1. 吸引人的職位標題
2. 職位亮點（3-5 點）
3. 具體工作內容
4. 必要條件 vs 加分條件
5. 公司福利與發展機會

語調要專業且吸引人，突出成長機會。
```

### 💬 **面試問題設計**
```
請為 [職位名稱] 設計一套面試問題：

職位層級：[初階/中階/高階]
關鍵技能：[列出 3-5 項核心技能]
團隊特色：[團隊文化或特殊要求]

請設計：
1. 開場破冰問題（2 題）
2. 技能評估問題（5 題）
3. 情境題/案例題（3 題）
4. 文化適配問題（2 題）
5. 候選人提問環節引導

每題都要附上：
- 評分標準
- 理想答案方向
- 追問重點
```

### 📈 **績效評估報告**
```
請幫我撰寫員工績效評估報告：

員工資訊：
- 姓名：[員工姓名]
- 職位：[職位名稱]
- 評估期間：[時間區間]

具體表現：
[描述工作表現、完成專案、團隊協作等]

請整理成：
1. 整體評分建議
2. 優秀表現亮點（3-5 項）
3. 需要改善的領域（2-3 項）
4. 發展建議與訓練需求
5. 下期目標設定

語氣要客觀公正，既肯定優點也指出成長空間。
```

## 📱 社群媒體 / 行銷 Prompt

### 🎨 **社群貼文創作**
```
請為 [品牌名稱] 創作社群媒體貼文：

平台：[Facebook/Instagram/LinkedIn 等]
目標受眾：[描述目標客群]
貼文主題：[產品推廣/活動宣傳/知識分享 等]
具體內容：[要傳達的訊息]
品牌調性：[專業/親切/創新/幽默 等]

請提供：
1. 吸睛的開頭句
2. 完整貼文內容（適合平台字數限制）
3. 相關 hashtag 建議（10-15 個）
4. 互動引導句（鼓勵留言/分享）
5. CTA 行動呼籲

請創作 3 個不同風格的版本供選擇。
```

### 📧 **EDM 郵件撰寫**
```
請撰寫一封行銷 EDM：

郵件類型：[產品介紹/促銷活動/會員通知等]
目標：[提升銷量/增加註冊/活動報名等]
受眾：[新客戶/老客戶/VIP會員等]

產品/活動資訊：
[詳細描述產品特色或活動內容]

請包含：
1. 吸引人的主旨（3 個選項）
2. 個人化開頭問候
3. 價值主張（為什麼要關注）
4. 具體優惠或好處
5. 明確的 CTA 按鈕文案
6. 緊迫感營造
7. 專業的結尾

避免過度推銷，重點放在客戶價值。
```

### 📊 **內容日曆規劃**
```
請為 [品牌名稱] 規劃一個月的內容日曆：

品牌特色：[描述品牌定位]
目標受眾：[客群描述]
主要平台：[列出使用的社群平台]
重要日期：[節日、活動、產品上市等]
內容主軸：[教育、娱樂、推廣等比例]

請規劃：
1. 每週主題設定
2. 貼文類型分配（圖文/影片/限動等）
3. 發文頻率建議
4. 互動活動安排
5. 重要節點的特別企劃

請用表格形式呈現，包含日期、主題、內容類型、平台。
```

```
4. 獲利能力評估
5. 現金流狀況
6. 風險警示
7. 改善建議

```

```

去年執行狀況：[去年預算 vs 實際支出]
今年目標：[業務目標或成長計畫]
特殊考量：[新專案、設備採購、人事異動等]

請協助：
1. 預算科目分類建議

4. 分季度預算分配
5. 控制點設定
6. 預算執行監控建議

請提供 Excel 表格格式建議。
```

## 🎯 專案管理 Prompt

### 📅 **專案計畫制定**
```
請幫我制定專案計畫：

專案名稱：[專案名稱]
專案目標：[具體要達成的結果]
預計時程：[開始和結束時間]
團隊成員：[參與人員及角色]
預算限制：[預算範圍]
主要限制：[時間、資源、技術等限制]

請提供：
1. 專案範圍定義
2. 工作分解結構（WBS）
3. 里程碑設定
4. 風險識別與對策
5. 溝通計畫
6. 品質管控要點
7. 成功評估標準

請用甘特圖概念安排時程。
```

### ⚠️ **風險評估報告**
```
請幫我進行專案風險評估：

專案背景：[專案簡介]
已識別風險：
[列出已知的潛在風險]

請分析：
1. 風險分類（技術/市場/資源/時程等）
2. 風險發生機率評估（高/中/低）
3. 影響程度分析（嚴重/一般/輕微）
4. 風險優先順序排列
5. 預防措施建議
6. 應急計畫
7. 風險監控指標

請用風險矩陣方式呈現評估結果。
```

## 🛠️ 進階使用技巧

### 🔧 **Prompt 優化方法**

#### ⭐ **提升準確性的技巧**
- **分步驟指令**：將複雜任務拆解成小步驟
- **範例示範**：提供期望輸出的範例
- **角色設定**：讓 AI 扮演專業角色
- **格式規定**：明確指定輸出格式

#### 🎯 **常用的 Prompt 結構**
```
角色設定 + 任務描述 + 具體要求 + 輸出格式 + 注意事項
```

#### 📊 **效果評估方法**
- **A/B 測試**：同一需求嘗試不同 Prompt
- **結果比較**：評估輸出品質和實用性
- **持續優化**：根據使用經驗調整 Prompt

### 💎 **高級應用場景**

#### 🔄 **多輪對話策略**
```
第一輪：基礎版本生成
第二輪：「請根據以下反饋調整...」
第三輪：「請進一步優化...的部分」
```

#### 📚 **知識整合應用**
```
請結合 [專業領域A] 和 [專業領域B] 的知識，
針對 [具體問題] 提供跨域解決方案...
```
